# ArgStem

The “Arg Stem” project is an interactive web platform, being
is an MVP, where students and people interested in STEM Careers can
benefit from resources, guides and modules that allow them to reinforce their knowledge,
obtain information about the careers in which they are interested, and stay up to date with the
topics from each of its areas.

To run the project, first install the dependencies, located in each of the folders, whether frontend or backend (you have to execute the command in both so that the dependencies can be installed) run npm install or npm i in the terminal as you see fit. , after having installed the dependencies in both folders, execute npm start in the frontend folder, to initialize the project.
